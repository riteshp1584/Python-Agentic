# Python-Agentic

Python-Agentic is a learning and experimentation repository focused on **Generative AI** and **LangChain**.  
It serves as a structured workspace to explore agentic workflows, build modular components, and document practical examples.

---

## 📌 Project Goals
- Understand the fundamentals of **LangChain** and its ecosystem.
- Experiment with **agent-based architectures** in Python.
- Preserve commit history while migrating learning modules (e.g., `LangChain_1`) into this dedicated repo.
- Provide a clean, organized foundation for future AI projects.
