## Title

**LangChain Prompt + OpenAI LLM Pipeline**

## Overview 

This script demonstrates a **basic LangChain workflow** that integrates a **prompt template** with an **LLM (OpenAI’s GPT model)** to generate structured responses. It shows how to define dynamic prompts, connect them to a language model, and run the chain to produce outputs. This serves as a simple yet powerful introduction to **prompt engineering with LangChain**.  

## Key Features  

- 📝 **Prompt Template**: Defines reusable templates with placeholders for dynamic inputs.  
- 🤖 **LLM Integration**: Uses `ChatOpenAI` to connect with OpenAI’s GPT models.  
- 🔗 **Chain Creation**: Builds an `LLMChain` that links the prompt and LLM together.  
- 🧪 **Test Run**: Executes a sample query to validate the pipeline.  


## Why It’s Useful  

- ✅ Provides a **starter template** for experimenting with LangChain.  
- ✅ Helps developers understand **prompt engineering basics**.  
- ✅ Serves as a **foundation** for building agentic workflows.  
- ✅ Useful for testing **OpenAI integration** before scaling to advanced use cases.
