Google Generative AI – Model Discovery Script

Overview

This script demonstrates how to connect to Google Generative AI (Gemini API) using an API key, configure the client, and list all available models that support the generateContent method. It provides a simple way to verify which models your API key has access to and ensures your environment is correctly set up for further development.

Key Features

🔑 Environment Setup: Loads the GOOGLE_API_KEY securely from .env.

⚙️ API Configuration: Configures the google.generativeai client with your key.

📋 Model Discovery: Lists all models accessible to your account that support generateContent.

✅ Quick Validation: Confirms your API key and environment are working correctly.

Why It’s Useful

✅ Provides a starter template for working with Google Generative AI.

✅ Helps developers quickly verify API connectivity and available models.

✅ Serves as a foundation for building applications with Gemini models.

✅ Ensures environment variables and API keys are correctly configured before moving to advanced workflows.
