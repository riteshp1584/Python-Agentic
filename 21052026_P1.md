Title

Local Ollama Agent with Sharpe Ratio Tool

Overview

This script demonstrates how to build a LangChain ReAct-style agent powered by Ollama (llama3.1:latest) with memory and a custom financial tool. It integrates a Sharpe Ratio calculator into the agent workflow, allowing the LLM to perform risk-adjusted portfolio analysis while maintaining conversational memory.

Key Features

🛠️ Custom Tool: Implements a calculate_sharpe_ratio function to evaluate portfolio performance.

🤖 LLM Integration: Uses ChatOllama with deterministic settings for reliable tool calls.

🧠 Memory Support: Employs MemorySaver to persist agent state across interactions.

🔗 Agent Creation: Compiles tools, LLM, and memory into a ReAct agent via create_agent.

📊 Streaming Execution: Streams agent outputs step-by-step for transparency during local runs.
